📊 Customer Churn Prediction Web App
A machine learning-powered web application built with Flask that predicts whether a telecom customer is likely to churn based on various features like contract type, monthly charges, internet service, etc.

🚀 Demo
🖥️ Coming soon: You can host this on Render or Railway to get a live link here

🧠 Features
Predicts customer churn using Logistic Regression

Clean UI using HTML + CSS + dropdowns/checkboxes

Dynamic prediction based on user input

Shows probability & result (Yes/No)

Trained with MinMaxScaler and saved model (.pkl) loaded during runtime

⚙️ Tech Stack
Area	Tools Used
Backend	Flask, Python
Machine Learning	Scikit-learn (Logistic Regression)
Frontend	HTML, CSS
Deployment	GitHub (ready), Render (optional)
Libraries	Pandas, NumPy, Joblib

📝 Dataset Used
Telco Customer Churn Dataset (Kaggle)
